# snake_diy

以 Python3 開發一個 貪食蛇遊戲，使用 pygame 作為遊戲引擎。

## requirement(User Story)

- [ ] 以 Python3 為開發語言
- [ ] 使用 pygame 作為遊戲引擎
- [ ] 使用英文顯示
- [ ] 具有排行榜功能
- [ ] 具有音效, 當蛇吃到食物時, 有音效
- [ ] 使用上下左右進行方向控制, 當方向與現在相反時不會有反應; 當相同時會稍微加速; 不同時轉向
- [ ] 貪食蛇的身體由多個方塊組成, 每吃到食物時, 會增加一個方塊
- [ ] 貪食蛇的身體會隨著方向而移動, 每移動一個方塊的距離, 會增加一分
- [ ] 當蛇撞到自己或邊界時, 遊戲結束, 並顯示目前的分數.
- [ ] 需要以獨立設定檔案來設定遊戲的設定, 例如蛇的顏色, 食物的顏色, 背景的顏色, 音效的音量, 蛇的移動速度, 邊界是否存在或者穿透
